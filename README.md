# Retail Sales Analysis – SQL Project

## Project Overview

This project analyzes retail sales transaction data using SQL to explore customer behavior, product category performance, sales trends, and key business insights. The project includes database setup, data cleaning, exploratory data analysis, and business-focused SQL queries.

## Tools Used

* SQL / MySQL
* Data Cleaning
* Aggregations
* CASE Statements
* Window Functions
* Business Analysis

## Database

**Database Name:** `retail_sales_db`
**Table Name:** `retail_sales`

The dataset includes transaction details such as transaction ID, sale date, sale time, customer ID, gender, age, product category, quantity, price per unit, cost of goods sold, and total sale amount.

## Project Objectives

* Create and structure a retail sales database.
* Clean the dataset by identifying and removing missing values.
* Perform exploratory data analysis to understand sales, customers, and categories.
* Answer business questions using SQL queries.
* Generate insights related to customer behavior, product performance, and sales trends.

## Business Questions Answered

* How many total transactions are available in the dataset?
* How many unique customers are included?
* Which product categories are available?
* Which categories generated the highest sales?
* Which customers contributed the most revenue?
* Which month had the highest average sales?
* How do transactions differ by gender and product category?
* How many orders were placed during morning, afternoon, and evening shifts?

## Key SQL Techniques Used

* `COUNT`, `SUM`, `AVG`
* `GROUP BY` and `ORDER BY`
* `WHERE` filtering
* `CASE` statements
* Date and time functions
* `RANK()` window function
* Customer and category segmentation

## Key Insights
* Compared sales performance across product categories and customer groups.
* Identified high-value purchases and top revenue-generating customers.
* Examined monthly sales patterns and order timing by shift.
* Translated transaction data into practical insights for KPI tracking and business decisions.

## Business Value

The analysis helps understand sales performance, customer behavior, product demand, and transaction patterns. These insights can support KPI tracking, targeted marketing, inventory planning, and performance improvement.
